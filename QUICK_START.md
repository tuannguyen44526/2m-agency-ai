## LẦN ĐẦU (Chỉ làm 1 lần)

### Bước 1 — Lấy API Key từ Anthropic
1. Mở: **https://console.anthropic.com/keys**
2. Đăng nhập (hoặc tạo tài khoản miễn phí)
3. Click **"Create Key"** → Copy key (bắt đầu bằng `sk-ant-...`)
4. Nạp tiền: **$5** là đủ dùng cả năm với quy mô 2M

### Bước 2 — Cài đặt (nếu chưa có Python)
1. Tải Python: **https://python.org/downloads** (click nút vàng to)
2. Cài đặt → **TICK CHỌN "Add Python to PATH"** ← quan trọng
3. Double-click file **`1_SETUP.bat`** → đợi 1 phút

---

## MỖI LẦN DÙNG

1. Double-click **`2_RUN.bat`**
2. Browser tự mở → **http://localhost:8501**
3. Paste API Key vào ô bên trái
4. Gõ lệnh → Click **🚀 Chạy Agency**
5. Đợi 30-60 giây → Toàn bộ nội dung xuất hiện
